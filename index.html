Modern software developement demands one's versatility working on multiple layers in the production. A full stack developer has the knowledge on all layers in Three-Tier Architecture, namely, presentation layer, business logic layer and data access layer.

I'm most interested in full stack development with JavaScript and Node.js, and have built an entire application for a restaurant from front-end presentation to back-end logic all by myself. My past experience also includes building and configuring RESTful API, handling HTTP request, client-server communcation, working with Node.js server and NoSQL database, MongoDB.

Front-End Development

I have knowledge on Human-Computer Interactions, UX/UI, prototyping and so on. I also have experience in building efficient front-end. With solid foundations on front-end triplets, HTML 5, CSS 3 and JavaScript, I have developed many applications. Ristorante Con Fusion

Ristorante Con Fusion Web App introduced a collection of lipsmacking dishes that tickle people’s culinary senses. People can use this web app to view promotions, browse dishes by category, comment and rate for each dish, filter, and submit feedback.

Flux Architecture was used to organize the application. It mainly took advantages of the modern React framework to manage various components and Redux store to maintain states.

Key Word: JavaScript, React, Flux, Redux, Thunk

Collaborative Online Judge

Collaborative Online Judge is a full stack project. It combined features of Google doc and Leetcode, made team-mode coding practice possible. A user can join or create room for each questions. It also supported code compiling execution and result judgement after a submission.

The project mainly consisted three part. OJ-Client, OJ-Server and OJ-Executor. The front-end interface was mainly built with AngularJS; The back-end server was managed by Node.js and Express, used MongoDB to problem data and Redis used to cached unsubmitted code. A user-code service was built with Flask to help execute code in the docker environment.

Key Word: AngularJS, Node.js, Express, MongoDB, RESTful API, Nginx, Docker

Real Time News

In this project, I designed and implemented a real-time news app based on scraping and recommendation system. The system uses a news pipeline to scrape latest news from various of sources such CNN, BBC. To render the news, I built a single-page web application using React. In addition, in order to customize news list for users, I designed and built a training pipeline for news topic modeling using TensorFlow.

Key Word: React, Node.js, Python, Web Scraping, Recommendation System
